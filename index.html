Problem System Data Pilot Team

Behavioral health access, rebuilt for the real world

We help care teams find plausible behavioral-health options without pretending broken directories are reliable.

The project combines public provider, facility, shortage-area, and payer data into a ranked access navigator for New Jersey. It is built for coordinators, discharge planners, and social workers who need a practical starting point, not another dead-end list.

Focus New Jersey statewide
Primary users Care coordinators and navigators
Output Ranked options with confidence bands

Why this matters

Referrals fail because the data is fragmented, stale, and hard to trust.

Directories drift

Provider lists go stale fast. Addresses move, specialties are messy, and network status can contradict across sources.

Teams need plausibility

Social workers and care coordinators need a defensible shortlist that explains why an option is worth trying first.

Shortage makes friction worse

When behavioral-health capacity is already constrained, navigation failure becomes its own barrier to care.

What the system does

A transparent navigator, not a magical booking engine.

01

Collect inputs

ZIP code, payer, care need, age group, telehealth preference, and travel radius.

02

Reconcile evidence

We match providers and facilities across public sources, then score them for fit, distance, and corroboration.

03

Return ranked options

Every result includes explanation tags and a confidence level so the user knows what the data supports.

Data backbone

Public data, fused carefully.

NPPES

Provider identity, taxonomy, addresses, and monthly churn.

SAMHSA N-SUMHSS

Behavioral-health facility services, locations, and utilization context.

HRSA HPSA and AHRF

Shortage-area designations and regional access context.

Payer Directory APIs

Audited network evidence from a limited set of public payer endpoints.

CMS IPF Quality

Facility quality signals where inpatient psychiatric reporting is available.

Source transparency

Every recommendation is grounded in visible evidence instead of opaque scoring claims.

Pilot path

Start in New Jersey, prove usefulness, then expand with discipline.

Initial pilot

Statewide coverage with a Camden-centered demo narrative for community relevance and tighter validation.

Deliverable

A working navigator for coordinators, plus a reproducible data pipeline, responsible-AI memo, and evaluation plan.

Operating principle

Show what the evidence supports. Flag uncertainty early. Never oversell live availability that public data cannot confirm.
