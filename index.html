New Jersey behavioral health navigation

Better referrals start with better data.

Care coordinators, social workers, and discharge planners deserve more than stale directories and dead-end phone calls. We fuse five public data sources into a transparent, ranked navigator that shows which behavioral health options are actually plausible — and explains why.

The problem

Referrals fail because the data underneath them is broken.

Every day, navigators and coordinators waste hours calling numbers from directories that were wrong before they were even published.

Directories drift

Provider addresses move, specialties are miscoded, and network status contradicts across sources. The directory was already wrong when someone printed it.

Navigators need plausibility, not promises

Social workers need a defensible shortlist that explains why an option is worth trying first — not a wall of names with no signal.

Shortage amplifies friction

When behavioral health capacity is already constrained, every failed referral attempt costs time a patient may not have.

How it works

A transparent navigator, not a black box.

01

User inputs

ZIP code, insurance type, care need, age group, telehealth preference, and travel radius. Simple inputs that match how navigators already think about referrals.

02

Evidence reconciliation

We match providers and facilities across five public sources, then score each option for service fit, geographic proximity, network evidence, and source corroboration.

03

Ranked results with explanation

Every result carries a confidence label — high, medium, or low — plus explanation tags showing exactly which sources support it and where evidence is thin.

High confidence 2+ corroborating sources, strong service and network fit
Medium confidence One strong source, partial corroboration
Low confidence Single source, weak network or service evidence

Data backbone

Five public sources. Zero black boxes.

Every recommendation traces back to visible, auditable evidence. No proprietary data, no opaque scoring.

CMS

NPPES

Provider identity, taxonomy codes, practice addresses, and monthly update churn tracking.

SAMHSA

N-SUMHSS

Behavioral health facility locations, service types, and utilization context.

HRSA

HPSA / AHRF

Mental health shortage area designations and county-level access context.

CMS

Payer directories

Audited network evidence from publicly mandated payer directory API endpoints.

CMS

IPF Quality

Inpatient psychiatric facility quality measures where CMS reporting is available.

Pilot scope

Start in New Jersey. Prove usefulness. Then expand.

Geography

New Jersey statewide coverage with a Camden-centered demo for community relevance and tighter validation against local ground truth.

Deliverables

Working navigator, reproducible data pipeline, responsible-AI documentation, evaluation plan, and a product story ready for funding conversations.

Principles

Show what evidence supports. Flag uncertainty. Never oversell live availability that public data cannot confirm.
